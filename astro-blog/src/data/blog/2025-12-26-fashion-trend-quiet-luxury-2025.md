## 1. 올드머니룩이란?

### 조용한 럭셔리의 정의

```
👔 올드머니룩 (Old Money Look)

【정의】
├─ 올드머니(전통 상류층) + 룩(스타일)
├─ 과시 X, 절제 O
├─ 품위와 교양 드러내기
└─ 로고 없는 고급스러움

【다른 이름】
├─ 콰이어트 럭셔리 (Quiet Luxury)
├─ 스텔스 럭셔리 (Stealth Luxury)
└─ 조용한 명품

【핵심】
"비싸 보이지만 티 안 내기"
```

*출처: [나무위키](https://namu.wiki/w/올드머니%20룩)*

### 올드머니 vs 뉴머니

| 구분 | 올드머니 | 뉴머니 |
|------|----------|--------|
| **로고** | 없음/최소화 | 크고 화려 |
| **색상** | 무채색, 베이지 | 원색, 화려 |
| **소재** | 캐시미어, 실크 | 다양 |
| **악세사리** | 진주, 심플 | 브랜드 과시 |
| **메시지** | "티 안 나는 부자" | "내가 부자다" |

## 6. 올드머니룩 실전 코디

### 상황별 스타일링

```
👗 올드머니룩 코디 가이드

【출근룩】
├─ 화이트 셔츠 + 슬랙스
├─ 카멜 코트
├─ 로퍼
└─ 진주 귀걸이

【캐주얼】
├─ 캐시미어 니트 + 청바지
├─ 로퍼 or 스니커즈
├─ 토트백
└─ 심플 시계

【데이트】
├─ 실크 블라우스 + 스커트
├─ 트렌치코트
├─ 힐 or 로퍼
└─ 진주 목걸이

"기본에 충실, 소재로 승부"
```

### 계절별 필수템

| 계절 | 핵심 아이템 |
|------|------------|
| **봄** | 트렌치코트, 리넨 셔츠 |
| **여름** | 린넨 원피스, 스트라이프 |
| **가을** | 캐시미어 니트, 울 코트 |
| **겨울** | 캐시미어 코트, 머플러 |

## 7. 가성비 올드머니룩

### 저렴하게 따라하기

```
💰 가성비 올드머니룩

【추천 브랜드】
├─ 유니클로 (캐시미어 라인)
├─ COS (H&M 프리미엄)
├─ 자라 (베이직 라인)
├─ 무신사 스탠다드
└─ 아울렛 명품

【핵심 전략】
├─ 기본 아이템 투자
├─ 무채색 위주
├─ 핏 중시
├─ 소재 확인
└─ 악세사리로 포인트

【예산 가이드】
├─ 니트: 5~10만원
├─ 셔츠: 3~7만원
├─ 슬랙스: 5~10만원
├─ 코트: 15~30만원
└─ 로퍼: 10~20만원

"비싸지 않아도 고급스럽게"
```

| **COS** | 중저가 | 코트, 슬랙스 |
| **무신사 스탠다드** | 저가 | 기본템 전체 |
| **자라** | 저가 | 셔츠, 바지 |
