## 2. 펫휴머니제이션

### 반려동물의 가족화

```
👨‍👩‍👧 펫휴머니제이션 (Pet Humanization)

【정의】
├─ 반려동물을 인간 가족처럼 대우
├─ '키우다' → '함께 살다'
├─ 소비 패턴 인간화
└─ 감정적 교류 중시

【소비 변화】
├─ 사료 → 휴먼그레이드 푸드
├─ 집 → 반려견 전용 인테리어
├─ 장난감 → 지능형 IoT 기기
├─ 병원 → 종합 헬스케어 센터
└─ 묘지 → 프리미엄 장례 서비스

【심리적 요인】
├─ 저출산 → 반려동물 = 자녀
├─ 1인가구 증가
├─ 정서적 위안 추구
└─ MZ세대 가치관

"반려동물도 가족이니까"
```

## 4. 강아지 vs 고양이

### 양대 시장 비교

```
🐕 vs 🐈 강아지 vs 고양이

【인구 현황】
├─ 강아지: 602만 마리
├─ 고양이: 313만 마리
├─ 강아지 가구: 535만 가구
└─ 고양이 가구: 270만 가구

【소비 패턴】
강아지 집사:
├─ 미용/의류 지출 높음
├─ 외출/산책 용품
├─ 사회화 훈련 투자
└─ 월평균 21.2만 원

고양이 집사:
├─ 사료/간식 프리미엄
├─ 캣타워/스크래처
├─ 건강검진 빈도 높음
└─ 월평균 16.8만 원

【트렌드 차이】
├─ 강아지: 펫캉스, 동반 외식
├─ 고양이: 스마트 급식기, 캣휠
└─ 공통: 프리미엄 사료
```

## 5. 펫캉스 트렌드

### 반려동물과 함께하는 여행

```
✈️ 펫캉스 (Pet + Vacance)

【정의】
├─ 반려동물 + 바캉스
├─ 동반 여행 문화 확산
├─ MZ세대 주도
└─ 코로나 이후 급증

【인기 시설】
├─ 펫 프렌들리 호텔
├─ 반려견 동반 펜션
├─ 애견 카페/레스토랑
├─ 펫 전용 수영장
└─ 반려동물 동반 캠핑장

【인기 지역】
├─ 강원도: 펜션 밀집
├─ 제주도: 펫프렌들리 확대
├─ 경기도: 당일 여행
├─ 부산: 해변 산책
└─ 가평/양평: 근교 인기

"우리 막둥이 빼고 여행? 절대 안 돼"
```
