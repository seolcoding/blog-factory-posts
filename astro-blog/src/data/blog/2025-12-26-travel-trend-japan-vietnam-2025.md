```

├─ 비행시간: 2~3시간
├─ 항공권: 20~40만원
├─ 주말여행 가능

└─ 결제 편리
```

## 3. 베트남 급부상

### 가성비 + 휴양

```
🇻🇳 베트남 인기 상승

【푸꾸옥】 🔥
├─ "베트남의 몰디브"
├─ 리조트 가성비
├─ 해변 + 수영장
└─ 신혼여행 인기

【다낭 & 호이안】
├─ 저렴한 물가
├─ 맛있는 음식 (쌀국수, 반미)
├─ 아름다운 해변
├─ 호이안 = 유네스코 세계유산
└─ 바나힐 골든브릿지

【나트랑】
├─ 휴양 리조트
├─ 스노클링/다이빙
├─ 저렴한 스파
└─ 러시아인 많음

"가성비 휴양의 정석"
```

### 베트남 여행 비용

| 항목 | 예상 비용 |
|------|----------|
| **항공권** | 30~50만원 |
| **숙소 (1박)** | 3~10만원 |
| **식사 (1끼)** | 5,000~15,000원 |
| **마사지 (1시간)** | 15,000~30,000원 |
| **5박 총 비용** | 80~150만원 |
