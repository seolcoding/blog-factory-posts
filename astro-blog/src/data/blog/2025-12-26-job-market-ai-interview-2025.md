## 2. AI 면접의 확산

### AI 면접이란?

```
🤖 AI 면접 평가 방식

분석 대상:
├─ 표정 (Facial Expression)
├─ 음성 (Voice Tone)
├─ 답변 내용 (Content)
├─ 발음 정확도
└─ 키워드 포함 여부

평가 결과:
├─ 종합 점수
├─ 역량별 분석
└─ 개선 포인트

"AI가 모든 것을 분석한다"
```

*출처: [링커리어](https://community.linkareer.com/employment_data/4069004)*

### AI 면접 합격 팁

| 항목 | 팁 |
|------|-----|
| **발음** | 또박또박, 적절한 속도 |
| **키워드** | 직무 관련 용어 필수 |
| **표정** | 자연스러운 미소 |
| **시선** | 카메라 정면 응시 |
| **답변** | 두괄식, 핵심 먼저 |

> "AI는 발음을 분석하기 때문에 또박또박 말하는 것이 핵심"

## 4. ATS(지원자 추적 시스템) 통과법

### AI가 먼저 본다

```
🔍 ATS (Applicant Tracking System)

역할:
├─ 자동 이력서 필터링
├─ 키워드 매칭
├─ 자격 요건 체크
└─ 점수화

현실:
"사람이 보기 전에 AI가 먼저 걸러낸다"

통과 확률 높이려면:
├─ 직무 연관 키워드 필수
├─ 기술명, 툴 구체적 기재
├─ 성과 지표 수치화
└─ 표준 포맷 사용
```

### ATS 통과 키워드 예시

| 직무 | 필수 키워드 |
|------|-------------|
| **개발자** | Python, React, AWS, CI/CD |
| **마케터** | ROI, CTR, GA4, 그로스해킹 |
| **기획자** | PRD, 와이어프레임, A/B테스트 |
| **디자이너** | Figma, UI/UX, 프로토타입 |

## 8. AI 시대 취업 생존 전략

### 필수 역량 체크리스트

```
✅ 2025 취업 필수 역량

기본:
☐ AI 툴 활용 능력 (ChatGPT, Copilot 등)
☐ 데이터 분석 기초
☐ 디지털 리터러시

직무별:
☐ 직무 관련 자격증
☐ 포트폴리오/프로젝트
☐ 인턴/경험

소프트스킬:
☐ 문제 해결 능력
☐ 커뮤니케이션
☐ 자기주도 학습
```

### 취준생 마인드셋

| DO | DON'T |
|-----|-------|
| AI 툴 적극 활용 | AI에 100% 의존 |
| 경험 쌓기 (인턴 등) | 스펙만 쌓기 |
| 직무 역량 증명 | 범용 자소서 |
