## 설정 방법

<HighlightBox
  type="info"
  title="iPhone에서 Apple Intelligence 활성화"
>
```
1. 설정 앱 열기
2. "Apple Intelligence 및 Siri" 탭
3. "Apple Intelligence" 토글 활성화
4. 언어를 "한국어"로 설정
```
</HighlightBox>

<HighlightBox
  type="warning"
  title="언어 설정 확인"
>
Apple Intelligence를 사용하려면 **Siri와 기기 언어가 모두 한국어**로 설정되어 있어야 합니다.

```
설정 → 일반 → 언어 및 지역
├── iPhone 언어: 한국어
└── Siri 언어: 한국어 (대한민국)
```
</HighlightBox>

## 자주 묻는 질문

<HighlightBox
  type="warning"
  title="Q: iPhone 14에서 사용 가능?"
>
**A**: 아니요. **iPhone 15 Pro 이상** 또는 **iPhone 16 전 모델**에서만 지원됩니다.
</HighlightBox>

<HighlightBox
  type="success"
  title="Q: 인터넷 없이 사용 가능?"
>
**A**: 네, 대부분의 기능은 **온디바이스**로 작동합니다. 단, 일부 복잡한 요청은 인터넷이 필요합니다.
</HighlightBox>

<HighlightBox
  type="info"
  title="Q: 한국어 Siri가 ChatGPT처럼 대화 가능?"
>
**A**: 현재는 제한적입니다. 고급 대화 기능은 2026년 이후 출시 예정입니다.
</HighlightBox>

<HighlightBox
  type="success"
  title="Q: 무료인가요?"
>
**A**: 네, **Apple 기기 구매자에게 무료**로 제공됩니다.
</HighlightBox>
