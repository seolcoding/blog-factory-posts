## 1. GMG / HMH - 가장 핫한 신조어

### GMG = 가면 가?

```
📱 GMG 사용 예시

친구 A: "강남 그 카페 GMG?"
친구 B: "ㅇㅇ GMG"
친구 A: "ㅇㅋ 30분 뒤 봐"

번역:
A: "강남 그 카페 (지금) 가면 가?"
B: "응 가면 가"
A: "알겠어 30분 뒤 봐"
```

### HMH = 하면 해?

| 표현 | 의미 | 사용 상황 |
|------|------|----------|
| **GMG** | 가면 가? | 약속 제안 |
| **HMH** | 하면 해? | 행동 제안 |
| **BMB** | 보면 봐? | 영화/드라마 제안 |
| **MMM** | 먹으면 먹어? | 식사 제안 |

> "GMG/HMH는 15~24세 연령대에서 가장 많이 사용됩니다"
> — [아보카도자이언트 MZ 트렌드 리포트](https://avocadogiant.com/newsletter/korea/2025-mz세대-신조어-밈/)

![친구 약속](https://media.giphy.com/media/xT9IgG50Fb7Mi0prBC/giphy.gif)
*GMG 하고 바로 만나는 MZ세대*

## 2. 허거덩거덩스 👴

### 인스타그램에서 시작된 밈

```
🎭 허거덩거덩스란?

기원: 인스타그램 채널 '잇츠쿨띵'

특징:
├─ 왜곡된 비율의 아저씨 이미지
├─ 처음 들어보는 워딩
├─ 수많은 패러디 양산
└─ 중독성 있는 리듬감

"허거덩거덩스~ 허거덩거덩스~"
```

### 패러디 유형

| 유형 | 설명 |
|------|------|
| **원본** | 아저씨 이미지 + 음성 |
| **동물 버전** | 강아지, 고양이 버전 |
| **연예인 버전** | 아이돌, 배우 얼굴 합성 |
| **상황 버전** | 직장, 학교 상황 패러디 |

> 왜 유행하는지 설명할 수 없지만, 한번 보면 중독됩니다

## 3. 김냄비펌피럽 🐱

### 고양이 집사의 즉흥 노래

```
🎵 김냄비펌피럽의 탄생

집사: (고양이랑 놀면서)
      "김냄비~ 펌피럽~♪"

고양이: "?"

X(트위터): 🔥🔥🔥

결과: 수백만 뷰 바이럴
```

### 왜 유행했나?

| 요소 | 설명 |
|------|------|
| **하이텐션 집사** | 에너지 넘치는 모습 |
| **고양이 무반응** | 대조되는 무심함 |
| **중독성 멜로디** | 한번 들으면 계속 흥얼거림 |
| **귀여움** | 고양이 = 정의 |

> 고양이 영상 + 중독성 노래 = 바이럴 공식

![고양이](https://media.giphy.com/media/mlvseq9yvZhba/giphy.gif)

## 4. 섹시푸드 (Sexy Food) 🍕

### 맛있어 보이는 음식의 신조어

```
🍔 섹시푸드 정의

섹시푸드 = Sexy Food

조건:
├─ 보기에 좋음 (비주얼)
├─ 맛있어 보임 (침샘 자극)
├─ 인스타그래머블
└─ 먹고 싶은 충동 유발

"이 버거 진짜 섹푸다"
= "이 버거 진짜 맛있어 보인다"
```

### 섹시푸드 예시

| 음식 | 섹푸 포인트 |
|------|-------------|
| **치즈 폭탄 버거** | 치즈 늘어남 |
| **탕후루** | 윤기 나는 코팅 |
| **수비드 스테이크** | 완벽한 미디움 레어 |
| **크로플** | 바삭한 와플 + 아이스크림 |

> "섹푸"는 MZ 친구들 사이에서 일상적으로 사용됩니다

## 6. 칠 가이 (Chill Guy) 😎

### 구글 선정 2025년 대표 밈

```
😎 칠 가이란?

이미지: 무심한 표정의 개 캐릭터
의미: "나는 쿨하니까 신경 안 써"

사용 예:
"시험 망쳤는데 나는 칠 가이라서 괜찮아"
"야근인데 나는 칠 가이라서 ㄱㅊ"

"내 멘탈은 칠 가이"
```

### 칠 가이 활용법

| 상황 | 칠 가이 반응 |
|------|-------------|
| 시험 망침 | "나는 칠 가이" |
| 연애 실패 | "칠 가이라서 ㄱㅊ" |
| 돈 없음 | "칠하게 버텨야지" |
| 월요일 | "칠 가이 마인드로" |

## 7. 이탈리안 브레인롯 🇮🇹

### 뇌를 썩게 만드는 밈

```
🍝 이탈리안 브레인롯

정의: 이탈리아 관련 요소로
      뇌가 녹는 느낌의 콘텐츠

예시:
├─ 과장된 이탈리아 억양
├─ 스파게티 먹방
├─ "마마미아~" 연발
└─ 피자 도우 던지기

"보면 볼수록 뇌가 녹아"
```

### 브레인롯 밈의 특징

| 특징 | 설명 |
|------|------|
| **반복성** | 같은 패턴 반복 |
| **무의미함** | 의미 없는 콘텐츠 |
| **중독성** | 계속 보게 됨 |
| **공유 욕구** | "이거 봐봐" |

> "브레인롯 콘텐츠는 생산성의 적입니다. 하지만 멈출 수가 없어요"

## 결론: 2025 밈 생존 가이드

### 핵심 신조어 정리

| 신조어 | 의미 | 사용 예 |
|--------|------|---------|
| GMG | 가면 가? | "홍대 GMG?" |
| HMH | 하면 해? | "운동 HMH?" |
