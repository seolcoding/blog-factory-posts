2025년 4월 1일, Apple Intelligence가 드디어 **한국어를 지원**하기 시작했습니다. iOS 18.4, iPadOS 18.4, macOS Sequoia 15.4에서 사용할 수 있는 모든 기능을 알아보겠습니다.

## 목차

## Apple Intelligence란?

Apple Intelligence는 Apple의 **개인 맞춤형 AI** 시스템으로, iOS 18, iPadOS 18, macOS Sequoia에 통합되어 있습니다.

### 핵심 철학

| 원칙 | 설명 |
|------|------|
| **개인화** | 사용자 맥락 이해 |
| **프라이버시** | 온디바이스 처리 우선 |
| **통합** | Apple 생태계 전반 연동 |

## 지원 기기

### iPhone

| 모델 | 지원 여부 |

### Mac

|------|----------|

## 주요 기능

### 1. Writing Tools (글쓰기 도구) ✍️

어디서든 텍스트를 선택하고 AI 도움을 받을 수 있습니다.

| 기능 | 설명 |
|------|------|
| **다시 쓰기** | 다른 톤/스타일로 변환 |
| **교정** | 맞춤법, 문법 검사 |
| **요약** | 긴 텍스트 압축 |

**사용 방법**:
1. 텍스트 선택
2. 팝업 메뉴에서 "Writing Tools" 선택
3. 원하는 기능 클릭

### 2. Smart Reply (스마트 답장) 💬

메시지와 메일에서 **한 번의 탭**으로 적절한 답장을 보낼 수 있습니다.

```
받은 메시지: "내일 회의 참석 가능하신가요?"

스마트 답장 제안:
├── "네, 참석 가능합니다."
├── "시간을 조율해야 할 것 같습니다."
└── "죄송합니다, 다른 일정이 있습니다."
```

### 3. 알림 요약 🔔

수많은 알림을 AI가 핵심만 요약해줍니다.

**Before**:
```
알림 1: 카카오톡 - 김철수: 점심 뭐 먹을래?
알림 2: 카카오톡 - 김철수: 아 그리고...
알림 3: 카카오톡 - 김철수: 오늘 날씨 좋다
알림 4: 카카오톡 - 김철수: 산책 갈까?
```

**After**:
```
📱 카카오톡 - 김철수 (4)
"점심 식사와 산책 제안"
```

### 4. Mail 정리 📧

이메일을 우선순위별로 자동 정리하고, 긴 이메일을 요약합니다.

| 기능 | 설명 |
|------|------|
| 우선순위 정렬 | 중요 메일 상단 표시 |
| 요약 | 긴 이메일 핵심 추출 |
| 빠른 답장 | 스마트 답장 제안 |

### 5. 사진 앱 기능 📸

| 기능 | 설명 |
|------|------|
| Clean Up | 사진에서 불필요한 객체 제거 |
| 자연어 검색 | "제주도에서 찍은 바다 사진" |
| 메모리 | AI 기반 추억 영상 생성 |

## 아직 지원되지 않는 기능

### 2026년으로 연기된 기능

| 기능 | 상태 | 예상 시기 |
|------|------|----------|
| 화면 인식 Siri | 연기 | 2026년 |
| 앱 내 액션 | 연기 | 2026년 |
| ChatGPT 통합 | 일부 제한 | 미정 |

### 한국어 미지원 기능

일부 기능은 현재 한국어로 사용할 수 없습니다:
- 일부 Siri 고급 기능
- 특정 언어 의존 기능

## 설정 방법

### iPhone에서 Apple Intelligence 활성화

```
1. 설정 앱 열기
2. "Apple Intelligence 및 Siri" 탭
3. "Apple Intelligence" 토글 활성화
4. 언어를 "한국어"로 설정
```

### 언어 설정 확인

Apple Intelligence를 사용하려면 **Siri와 기기 언어가 모두 한국어**로 설정되어 있어야 합니다.

```
설정 → 일반 → 언어 및 지역
├── iPhone 언어: 한국어
└── Siri 언어: 한국어 (대한민국)
```

## 프라이버시 보장

### 온디바이스 vs 클라우드

| 처리 방식 | 설명 | 예시 |
|---------|------|------|
| **온디바이스** | 기기 내 처리, 데이터 외부 전송 없음 | 알림 요약, 글쓰기 도구 |
| **Private Cloud Compute** | 필요시 암호화 클라우드 사용 | 복잡한 요청 |

Apple은 사용자 데이터를 서버에 저장하지 않으며, 암호화된 전송을 보장합니다.

## 자주 묻는 질문

### Q: iPhone 14에서 Apple Intelligence를 사용할 수 있나요?
A: 아니요. **iPhone 15 Pro 이상** 또는 **iPhone 16 전 모델**에서만 지원됩니다.

### Q: 인터넷 없이도 사용 가능한가요?
A: 네, 대부분의 기능은 **온디바이스**로 작동합니다. 단, 일부 복잡한 요청은 인터넷이 필요합니다.

### Q: 한국어 Siri가 ChatGPT처럼 대화할 수 있나요?
A: 현재는 제한적입니다. 고급 대화 기능은 2026년 이후 출시 예정입니다.

### Q: 무료인가요?
A: 네, **Apple 기기 구매자에게 무료**로 제공됩니다.

## 결론: Apple Intelligence의 현재와 미래

2025년 4월 한국어 지원은 시작에 불과합니다:

✅ **현재 가능**:
- Writing Tools
- Smart Reply
- 알림 요약
- 사진 Clean Up

⏳ **곧 출시 예정**:
- 화면 인식 Siri
- 앱 내 액션
- 더 강력한 ChatGPT 통합

Apple Intelligence는 "조용하지만 강력한" Apple 특유의 접근 방식으로 발전하고 있습니다. 프라이버시를 희생하지 않으면서 일상의 편의를 높여주는 것이 핵심 가치입니다.
