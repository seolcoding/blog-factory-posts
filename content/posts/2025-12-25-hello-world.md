## Blog Factory란?

Blog Factory는 AI를 활용한 자동 콘텐츠 생성 시스템입니다.

### 주요 기능

- 실시간 트렌드 분석 (Google Trends, Naver, DeepNews)
- SEO 최적화된 콘텐츠 자동 생성
- GitHub Pages 자동 배포

### 기술 스택

- **MCP Servers**: trendkit, deepnews, naver-search
- **Blog**: Hugo + PaperMod
- **Deployment**: GitHub Pages

앞으로 다양한 트렌딩 콘텐츠를 만나보세요!
